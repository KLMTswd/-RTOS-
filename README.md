# -RTOS-

集成**非接触式水位监测（0.5pF灵敏度）与WHO标准电解质饮料自制（±0.5g精度）**双功能。
通过机电一体化设计实现： 
🚰 科学饮水闭环：四段式超声水位检测（±2mL）+硬件级蜂鸣提醒（90dB）

🤖 自动化制饮：三自由度机械臂（0.5mm定位误差）+蠕动泵组（±3%流量精度）
